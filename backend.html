後台系統
辭典、出題、AI、任務分流

辭典和出題是兩套雲端系統：辭典負責權威字義與詞庫快取，出題負責題型資料、難度校準與 AI 生成。Nexora AI 讀取兩邊資料，安排任務與回饋。

服務分層 4 個後台服務
雲端辭典系統
Cloud Ready
優先查自建 Nexora 多益詞庫，再查授權字典 API 或外部來源連結。常見字快取到 Cloudflare KV，降低成本。
POST /api/dictionary/search GET /api/dictionary/cache/:word
雲端出題系統
Cloud Ready
依 Part 5 考點、難度、學生錯題紀錄生成題目。每題保存答對率、作答時間、錯誤選項分布。
POST /api/quiz/generate POST /api/quiz/attempt
Nexora AI 助理
Needs Model Key
讀取辭典結果、錯題資料、任務完成率與學習目標，產生個人化建議。模型金鑰只放 Worker / Vercel 環境變數。
POST /api/assistant POST /api/assistant/summary
學習任務系統
D1 Ready
存目標分數、每日任務、完成紀錄與 AI 推薦。前端目前可操作，正式上線接 Cloudflare D1。
GET /api/tasks POST /api/tasks
建議資料流 Cloudflare + Vercel
1
Dictionary
查字先走詞庫
自建詞庫命中就直接回覆；沒有命中才走授權字典或 AI 補充說明，並寫入待審詞條。
2
Quiz
出題先選考點
後台先選文法點與難度，再讓模型生成題目，最後由規則檢查答案、選項與 JSON 格式。
3
Nexora AI
用資料安排任務
AI 不只是回答問題，而是讀取查字、出題、錯題與任務紀錄，回推下一天的學習安排。
